Português:

"Neste projeto, a chave da API (OMDb) está visível no código front-end. Estou ciente de que, em um ambiente de produção real, chaves de API devem ser protegidas no lado do servidor (Back-end) ou através de variáveis de ambiente em serviços como Netlify/Vercel. Como este é um projeto educacional de Front-end hospedado no GitHub Pages, a chave foi mantida para garantir que a funcionalidade de busca funcione diretamente no navegador de quem visitar o site."

English:

"In this project, the API key (OMDb) is visible in the front-end code. I am aware that in a real production environment, API keys should be protected on the server-side (Back-end) or through environment variables in services like Netlify/Vercel. Since this is a Front-end educational project hosted on GitHub Pages, the key was kept to ensure that the search functionality works directly in the browser for anyone visiting the site."
